In order to strat the program, you need to run WeatherStation.java class in your prefered IDE. When the program runs user will be propmted to enter one of the three possible 
interfaces for displaying the tempereture readings. Those chocies will be a) AWT UI, b) SWING UI c) terminal. After entering the choice, press enter. After these steps, program 
will run the selected way of displaying the data and present user with different temperaute units and their corresponding values as they change. If the user enters an invalid 
choice, the program will display an error message indicating the wrong input and terminate. To correct your input, run the WeatherStation.java again and enter one of the valid 
choices.
